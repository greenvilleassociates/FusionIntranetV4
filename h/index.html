Joomla is installed in this directory already... if PHP indexing is enabled and put in first position... you'll get index.php first.
However your nginx.conf, Apache vhost, or IISWeb.config must be set to load index.php before index.html or index.htm which is not standard
This site requires PHP 7.4 or PHP 8.06 to run Joomla 3.10 which is included with this distribution.
You DO NOT WANT TO UPGRADE JOOMLA to 4.0 or 5.0 with this product as it removes native APIs which are required for advanced PLL(Page Level Locking)
